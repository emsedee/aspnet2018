# Torens van Hanoi

[Beschrijving op Wikipedia](https://nl.wikipedia.org/wiki/Torens_van_Hanoi)

Voor deze opdracht is het de bedoeling een klassenmodel verder te implementeren.
Je implementeert klassen met bijhorende members (properties, methoden, ...) in project
TorensVanHanoi.Model in de namespace TorensVanHanoi.Model
in de solution TorensVanHanoi.

Er zijn unit tests gemaakt met xUnit. Die unit tests zijn de specificatie van wat moet
geïmplementeerd worden in TorensVanHanoi.Model.

Na builden van de solution kan je in Test Explorer in Visual Studio zelf je punten bepalen.
Per groen bolletje krijg je 1 punt.
Per rood bolletje krijg je 0 punten. Het totaal is het totaal aantal unit tests in het project
**TorensVanHanoi.Test**.
